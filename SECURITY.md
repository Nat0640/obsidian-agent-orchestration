# Security Policy

## Supported Versions

Only the latest release of this project is actively supported with security updates.

## Reporting a Vulnerability

If you identify a security vulnerability, hardcoded credential, or local path leakage in this repository:
1. Please **do not** open a public GitHub issue.
2. Contact the maintainer directly via email: `developer@example.com` (please replace this in your fork with your actual contact address).
3. We will review the report and release a sanitized patch as soon as possible.
